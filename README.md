# Project Fuzzy Search — Case Study

A public case study of an 18-month enterprise multilingual smart knowledge retrieval system, delivered as Lead Project Manager and Product Owner.

**Live:** https://roxinch.github.io/project-fuzzy-search/

## About

This single-page case study documents the design, delivery, and impact of an NLP + Vector Search platform supporting Traditional Chinese, Simplified Chinese, English, and Korean for an enterprise customer service organization.

**Key outcomes**

- 85% retrieval-efficiency improvement (5 min → 45 sec)
- Search-failure rate reduced from 40% to 5%
- Full ROI within 6 months
- French module integration in 2 weeks (vs. 2-month estimate)
- 12-person cross-functional Scrum team across 4 departments
